## SETUP

You'll need:
  - Docker
  - Compatible image versions of Flink, Kafka, Grafana and Prometheus.

### Step 1:
Choose one of the applications, WC or kafka.

In a terminal inside the chosen directory, run:

```bash
docker compose up
```
